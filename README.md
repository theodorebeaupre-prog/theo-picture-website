# Théo Picture — Website

> Editorial design portfolio inspired by sutera.ch — "Creativity, By Lens."

## Stack
- Pure HTML, CSS, JavaScript (no framework)
- Space Grotesk + Space Mono (Google Fonts)
- Zero dependencies

## Features
- Blueprint/editorial layout (Sutera-inspired)
- Camera 3D tilt on mouse move
- Real-time Québec clock
- FR/EN language toggle
- Scroll reveal animations
- Custom crosshair cursor
- Fully responsive

## Structure
```
theo-picture-website/
├── index.html      # Main page
├── style.css       # All styles
├── script.js       # Interactions & animations
└── README.md
```
